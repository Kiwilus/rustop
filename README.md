# rustop
Neofetch alternative written in Rust.

---

## Features

- OS name
- Kernel
- CPU name
- RAM (used/total)
- System language
- batman ASCII banner

---

## Installation
1. clone the repo
```bash
git clone https://github.com/Kiwilus/rustop.git
```
2. change into the project directory
```bash
cd rustop
```
3. and install the tool
```bash
cargo install --path .
```
4. then execute with:
```bash
rustop
```
